# The Audiovisual Archive 🎞️🎵

**The 17-Month Synergy Challenge** is an interactive web platform designed to document a chronological journey through cinematic milestones and personal musical references.

## 🎯 The Concept
The project was born from a creative hypothesis: *Can we predict the perfect musical counterpart to a film based solely on its premise?*

Over 17 months, I paired one film from my **"Discovery Watchlist"** (movies I hadn't seen) with one **"Reference Album"** (personal favorites). After each viewing, I documented a "Synergy Verdict" to analyze if the predicted aesthetic vibe held true.

## 🚀 Technical Features
- **Vanilla Stack:** Built entirely with HTML5, CSS3, and JavaScript (No heavy frameworks).
- **Interactive UI:** A custom-built "Sticky Card Stack" for immersive navigation.
- **3D Interaction:** Implementation of 3D tilt effects and particle systems using **Three.js**.
- **Responsive Design:** Fluid layout optimized for both desktop and mobile experiences.
- **Micro-interactions:** Smooth LERP-based animations and custom UI feedback.

## 🛠️ Performance & Best Practices
- **Lerp Animation:** Smooth linear interpolation for the 3D tilt effect.
- **Hardware Acceleration:** Optimized using `will-change` and `transform-style: preserve-3d`.
- **Clean Architecture:** Modular JavaScript logic and organized CSS variables.

## 📖 How it Works
1. **The Discovery:** A seminal film selected for its historical or aesthetic importance.
2. **The Reference:** An essential album paired based on intuitive synergy.
3. **The Verdict:** A critical analysis of the audiovisual match.
